CSS specificity


¹ The mechanism of specificity in web browsers is employed to determine the most appropriate CSS declaration for an element. This process, in turn, helps to decide the corresponding property value to be assigned to that element. To determine which rule from conflicting CSS declarations takes precedence over others, the specificity algorithm computes the weight of a CSS selector.
